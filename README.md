# Street Map Project

## 怎样运行 ?
### 1.下载工程之后编辑index.html在[YOUR_API_KEY]位置添加你的google api key，保存。
### 2.用谷歌浏览器打开index.html后即可查看运行效果。


## 功能点介绍：
### 1.左侧菜单支持展开和关闭。
### 2.左侧筛选器支持筛选地区子集并将结果体现在列表和地图上。
### 3.点击左侧列表项或者右侧地图标记点后标记会以动画形式展示，并显示详细信息，再次点击动画展示停止。
### 4.位置详细信息支持查看评论跳转。
### 5.自定义地图样式。
### 6.knockout监听组件事件和属性。
### 7.返回结果未符合预期时的显示兼容性处理。
